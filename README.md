# TBgram
Spring Boot와 JPA를 활용한 회원 관리 시스템으로, 뉴스 피드 작성, 댓글 기능, 친구 관리 기능을 포함한 소셜 네트워크 서비스입니다.

📆 개발 기간 25.02.14 ~ 25.02.19
